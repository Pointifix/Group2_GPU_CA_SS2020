virtualbox with ubuntu
uname -r should NOT say Windows
install g++ or gcc
install latest version of cmake
	important to install manually to get the latest version 3.15!
	https://vitux.com/how-to-install-cmake-on-ubuntu-18-04/
install openssh-server and make sure it's up and running
vm > settings > network > Adapter 1 > Advanced > Por Forwarding

	Name	| Protocol	| Host IP	| Host port	| Guest IP	| Guest Port
	SSH		| TCP		| 127.0.0.1	| 2222		| *********	| 22
	
	*********	->	do "ip a" on Ubuntu and find out your port
					for me, it was 10.0.2.15
					
set up a remote host on clion (or whatever IDE supports this functionality...)
	File > Settings > Build, Execution, Deployment > Toolchains > + > Remote Host > GEAR near credentials
	
	Host		| Port	| User name	| Password
	127.0.0.1	| 2222	| *********	| ********
